# 《腌笃鲜》模组简介

> **检索标签**：[归属库：腌笃鲜] [数据类型：模组概述] [核心关联：原创人物, 原创怪物, 元辰劫运, 神话书说]

**腌笃鲜**是以《神话书说》为前置的非官方扩展、优化与修复模组。在保留神话世界主体体验的基础上，模组现已正式尝试重置人物与怪物篇章，并继续完善元辰劫运、法宝、角色机制和联机体验。

## 核心特性
- **腌笃鲜人物篇**：重置神话人物与全新玩法，首位人物为玉兔仙子
- **腌笃鲜怪物篇**：首位怪物为黑风大王·熊罴真身
- **元辰劫运系统**：核心机制十二元辰神通觉醒
- **四象三合局**：蛛皇形态的深度强化（第一阶段仅开放盘丝娘娘）
- **物品与法宝**：蒲团、寸心精魄、喜蛛天罗衣、吞天紫金葫芦、天地宝鉴、须弥玉净瓶、辟火罩等专属物品
- **火元淬炼系统**：武器和护甲的位面强化玩法
- **神话人物机制增强**：全神话角色的专属增强
- **体验优化与便利设施**：智能谷仓、百宝囊、八卦炉炼丹等
- **熊猫小镇焕新**：青竹小店升级，接入镇票、上鲜、盘账、立契、动态价格与甲木签玩法
- **聚宝盆革新**：重构聚宝盆抽奖体验，新增阶段奖池、分层保底、定愿气运与寸心机缘包
- **兼容与崩溃修复**：针对神话人物、常见物品技能与联机操作提供专项修复和适配

> **更新提醒**：聚宝盆革新与熊猫小镇焕新现由《腌笃鲜》主模组直接提供。模组配置彼此独立，壶天百戏中的旧开关不会转移，请在主模组设置中重新确认这两项开关。原“青竹小店焕新”与“人参果树：寻宝概率”配置已经移除，不再读取旧设置。

# 元辰劫运系统

> **检索标签**：[归属库：腌笃鲜] [数据类型：核心机制] [核心关联：寸心, 劫运, 星盘]

## 系统开启流程

元辰系统只能通过蒲团打坐呼出，玩家必经寻仙问道的流程：

1. **前往桃岛**：找到"灵台方寸山"，进入内室"阅读天书"（很多玩家阅读后没有解锁蒲团，是因为配置中元辰劫运系统开关未打开）
2. **制作蒲团**：在神话专属制作栏，消耗材料解锁并制作物品"蒲团"
   - 合成配方：`竹子` ×9 + `绳子` ×9
   - 科技需求：灵台方寸（阅读天书后解锁）
3. **打坐入定**：将蒲团放置在安全之处，右键点击进行"打坐"，元辰劫运图UI星盘展开

> **[机制警告]**：打坐期间移动（WASD）或被怪物攻击会强制退出。正常退出请按 ESC，临时暂停可使用服务器暂停键（默认P）。

## 核心概念

| 概念 | 说明 | 数值 |

| **劫运** | 经历神话相关事件积累劫运（达到26重劫运即可拥有12寸心，无需全部经历，顺其自然即可），达到阈值后可凝结寸心 | 总计35重 |

## 星盘结构

**左侧：元辰劫运图**
- **外圈（十二元辰）**：对应十二元辰星君（生肖）。点亮后获得"神通技能"
- **内圈（十二重楼）**：对应十二地支（子丑寅卯...）。点亮后利用"天时倍率"放大技能效果
- **双鱼按钮**：
  - 阳鱼（倾注寸心）：消耗寸心点数，激活星盘上的重楼或元辰
  - 阴鱼（凝结寸心）：历经劫运后，凝结寸心点

**右侧：灵台方寸境**
- **天时罗盘**：显示当前月相及推演"时辰"
- **数据面板**：显示当前【寸心点】和已历经的【劫运数】
- **技能预览**：鼠标悬停查看技能效果、触发概率、元辰代价及共振加成
- **箴言提示**：随机刷新未经历的"神话劫运事件"引导

## 急急如律令（洗点）

点错元辰可通过"急急如律令"重新选择。
- **代价**：消耗道具"急急如律令"
- **冷却**：40分钟现实时间（约游戏内5天）

# 共振倍率计算机制

> **检索标签**：[归属库：腌笃鲜] [数据类型：数值机制] [核心关联：共振, 倍率, 月相]

玩家只有12点寸心。全点外圈获得12个基础技能；选择内圈则技能威力几何倍增。

## 时间法则

十二重楼绑定月相，从新月到满月再到新月（约18天），十二地支依次轮转。一个"天时"持续约1.5天。（当下时辰可通过打坐在内景中查看）

## 倍率算法

| 共振层级 | 条件 | 效果 | 公式 |
| --- | --- | --- | --- |
| **本命共振** | 外圈点了某生肖，内圈也点对应地支（如外寅+内寅） | 该技能永久+25%效果 | `倍率 = 1.25` |
| **天时共振** | 当前天时运转到已点亮的重楼 | 所有已激活技能额外+50% | `倍率 = 1.50` |
| **极致共振** | 新月+满六阴重楼 或 满月+满六阳重楼 | 所有技能2.5倍（+150%） | `倍率 = 2.50` |

> **[机制说明]**：极致共振时，本命共振不再单独计算。共振倍率计算公式：`最终倍率 = 2.50`（极致共振）/ `1.75`（本命+天时共振）/ `1.25`（仅本命共振）/ `1.50`（仅天时共振）/ `1.00`（无共振）

## 阵营接轨

- **拥抱暗影**：点满6阴重楼（丑/卯/巳/未/酉/亥），自动加入暗影阵营
- **拥抱月亮**：点满6阳重楼（子/寅/辰/午/申/戌），自动加入月亮阵营
- 以首次满6属性为准，可通过"急急如律令"改换阵营，且只有神话人物与《腌笃鲜》玉兔仙子可获得阵营增强
- **玉兔仙子例外**：六阴不会为玉兔新锁定暗影阵营；六阳会锁定月亮阵营。若角色此前已经选择暗影，换成玉兔时选择会被保留但暂不生效，换回符合条件的人物后可继续恢复

手持莹月琵琶右键打开官方样式曲目轮盘，选曲后进入节奏演奏。

- 敌对减益曲与场域支援曲不掉落地面音符，效果直接作用于实际目标
- 已经获得的基础或满弦效果不会因玉兔停奏而回滚；神合属于演奏中的动态增强，停奏后立即消失

> 黑风大王败而化风，本欲借遁法脱身。若以吞天紫金葫芦先一步锁住战败之地，黑风便无处可逃，妖王被迫显出熊罴真身。

### 触发条件

熊罴真身不能通过普通刷新直接出现，必须完成以下流程：

1. 找到并挑战《神话书说》原黑风大王
2. 在其死亡前，提前释放吞天紫金葫芦的“吞天控场”
3. 必须等待吞天施法真正完成、活动场已经展开
4. 让原黑风大王在活动场覆盖范围内真正战败
5. 原黑风大王先按神话原流程完成死亡与掉落，随后生成的战败遁风会被吞天场截住
6. 遁风无法逃走被控制在吞天施法点，并在那里显化熊罴真身

**判定重点：**

- 先杀死黑风大王、再补放吞天无效
- 吞天仍在蓄力、活动场尚未真正建立时击杀无效
- 战败遁风从场外飘入吞天范围无效，判定只看原黑风死亡瞬间是否被活动场覆盖
- 偷蜜入场风、黑风大王超时离场风均不是战败遁风，不能触发真身
- 多个吞天场同时覆盖死亡点时，只由最近的一处取得触发权
- 同一时间只会存在一场熊罴真身遭遇

| :--- | :--- |
| 1次 | 24000 |
| 2次 | 21000 |
| 3次 | 18000 |
| 4次 | 15000 |
| 5次及以后 | 12000 |

- 单次必要的原黑风前置击杀不降档
- 计数在原黑风真正死亡时立即增加
- 熊罴真身生成后会锁定本次最大生命，战斗中或读档后不会重新按人数、当前次数计算

- 挑战失败、真身返风或世界重载不会清零计数；下次必须重新击杀原黑风触发，生命继续按上表下降，最低12000

- 黑旋生成前有1.25秒预警，生成后以低速追随本次锁定的目标2秒
- 每次伤害脉冲造成35普通伤害+5位面伤害
- 黑旋会绕开非法地形，不会因为轻微地形障碍立即消失
- 黑旋附近的石障每0.5秒被侵蚀1点耐久
- 同时存在数量随阶段提升：

| 阶段 | 生命区间 | 黑旋同时上限 |
| :--- | :--- | :--- |
| 第一阶段 | 100%～70% | 1 |
| 第二阶段 | 70%～35% | 2 |
| 第三阶段 | 35%以下 | 3 |

### 黑风倒卷与风王冲山

生命低于70%后，真身会把石障从普通掩体升级为机制解题资源。

1. 只有场上至少保有2座有效石障、没有尚未结束的黑旋，且存在位于真身超过10格、不超过14格处、未被石障遮挡并可被牵引的陆地玩家时，真身才会选择倒卷
2. 真身先以完整动作蓄积黑风，中心风压逐渐扩大；起手后不会因为初始目标跑位而取消整招
3. 正式吸入开始后，每次风压都会检查20格范围符合条件的玩家才会被快速拉向真身
4. 进入约3格风眼的玩家被捕获并暂时失去移动
5. **若至少捕获1人**：真身锁定被捕获者，保证完成一次普通挥击；实际伤害判定结束后释放所有被捕获玩家
6. **若最终无人被捕获**：真身随即进入风王冲山
7. 冲锋前会显示预警，玩家需要继续走位，把真身引向石障
8. 真身撞碎石障后进入6秒硬直；若撞中玩家，则造成110普通伤害+15位面伤害并击飞

- 骑乘玩家仍以玩家本体参与倒卷的起手、拉拽与捕获判定；保底挥击继续沿官方骑乘规则把伤害转移给有效坐骑，不会额外强制掉落坐骑

> **[机制要点]**：石障背风侧只解决“倒卷抓人”；倒卷挡住后仍要完成第二步，引导冲山撞石。提前把所有石障耗尽会让这一组合失去正解。

### 万字黑风吼

真身生命首次跨过70%与35%时，各强制进行一次万字黑风吼。若一次高伤同时跨过两个阈值，第二次会排队等待，至少间隔6秒，不会无操作地背靠背立刻爆发。

**爆发范围与伤害：**

- 判定范围为真身周围24格
- 冲击波造成15普通伤害+5位面伤害
- 每名玩家在爆发瞬间检查身体装备栏中的**锦襕袈裟**
- 只放在物品栏里无效；爆发后才换上也无法解除已经产生的控制

**穿着锦襕袈裟：**

- 完全免疫本次冲击波、硬控与后续普攻猎杀
- 获得本次技能期间的黑风拉力免疫，并显示佛光护体
- 结算中只要至少一名玩家成功护体，猎杀收束后真身会受到佛光反震，进入4秒输出窗口
- 成功破解后会短时间提高真身对袈裟玩家的仇恨权重，给团队其他成员创造空间

**未穿锦襕袈裟：**

- 先承受冲击波，再进入失去操作的心神控制
- 真身只以普通挥击逐个猎杀被控制玩家，不插入黑旋、敲地、冲锋或连击
- 同一玩家受到第2次真实普通攻击命中后解除控制
- 控制最多持续12秒；玩家死亡、真身撞石硬直、真身死亡或挑战失败也会提前解除

**石障保底：**

- 未穿袈裟但处于有效石障背风侧时，冲击波伤害降为35%，但仍会被控制
- 猎杀期间若真身高速追击撞上石障，会触发6秒硬直并终止本轮控制
- 石障不是锦襕袈裟的完全替代；无袈裟、无石障时，玩家会完整承受冲击波与至多两次普通攻击

### 阶段流程

| 阶段 | 生命区间 | 战斗变化 |
| :--- | :--- | :--- |
| **第一阶段** | 100%～70% | 学习敲地布障、基础连击与1个定点黑旋；重点是保存石障 |
| **70%检验** | 首次跨过70% | 强制万字黑风吼；之后解锁黑风倒卷与风王冲山 |
| **第二阶段** | 70%～35% | 黑旋同时上限提高到2，石障开始同时承担倒卷遮挡与冲山硬直 |
| **35%检验** | 首次跨过35% | 再次强制万字黑风吼 |
| **第三阶段** | 35%以下 | 黑旋同时上限提高到3，场地资源压力达到最高，但黑旋速度、寿命与单次伤害不再继续增强 |

### 挑战失败与返风

- 真身以自身实时位置为中心检查60格范围内的存活玩家
- 连续20秒没有任何存活玩家留在范围内，挑战判定失败
- 宽限期间只要有存活玩家重新进入60格，失败倒计时立即取消
- 失败后真身不会掉落任何奖励，会清理黑旋、石障、控制与技能状态，再次化作黑风离开
- 下一次挑战必须重新完成偷蜜、击杀原黑风、预铺吞天场并在场内完成击杀的完整流程
- 冻结、丢失当前攻击目标或短暂脱战不等于挑战失败，只看真身附近是否仍有存活玩家

### 击杀奖励

原黑风大王会先完成自身正常掉落，熊罴真身奖励在其后独立结算，两套战利品互不覆盖。

| 奖励 | 数量 | 多倍掉落 |
| :--- | :--- | :--- |
| **辟火罩** | 固定1件 | 不受任何普通或概率多倍掉落影响 |
| **熊皮** | 1张 | 普通掉落体系，可受合法多倍掉落影响 |

- 真身不额外掉落肉、宝石、寸心精魄、月亮材料、暗影材料或原黑风战利品
- 失败返风、异常移除或未完成真实击杀均不产生奖励
- 辟火罩的完整效果详见“物品与法宝”篇

# 高频Q&A
> **检索标签**：[归属库：腌笃鲜] [数据类型：常见问题] [核心关联：FAQ, 问题解答]

**Q1：为什么我在家砍树/打怪，理智和饥饿值突然狂掉？**
A：查查你的星盘！你肯定激活了【子鼠】（双倍掉落扣理智）或者【亥猪】（多倍战利品扣饥饿）。天之道，不白嫖。

**Q2：我点错了，点【散去修为】为什么没反应？**
A：这可是逆天改命的后悔药！冷却时间长达40分钟（5个游戏天数），请三思！

**Q3：为什么极阴/极阳之夜，我的伤害数字突然爆炸了？**
A：恭喜触发"极致共振"！凑齐6阴或6阳，并在新月/满月之夜，能力获得2.5倍独立乘区加成！

**Q4：我是盘丝娘娘，为什么我突然空蓝（没蛛丝值）了？**
A：你大概率激活了金局！金局拥有"绝对霸体"，每次受到硬控自动扣除5点蛛丝。被连续多段攻击时蛛丝会快速抽干，请注意拉扯！

**Q5：我有12点寸心，能把四个"三合局"全点亮吗？**
A：可以点亮四局基础，但如果想激活成局需要6点（3外圈+3内圈）。12点上限意味着最多只能捏出两个成局。

**Q6：为什么订阅启用后进不去游戏/直接崩溃了？**
A：最常见原因是没有订阅原版《神话书说》模组！本模组必须依赖原模组运行。

**Q7：玩着玩着突然报错/崩档了，怎么办？**
A：不要慌！先在模组设置中关闭该功能模块，即可恢复原版逻辑。同时恳请带上崩溃日志加群反馈。

**Q8：神话人物引发技能崩溃？**
A：神话人物相关功能需要同时适配官方更新、原有人物机制和其他人物模组。若仍发生技能崩溃，请记录所用人物、物品或动作、同时启用的模组，并携带崩溃日志加群反馈。

**Q9：有些新增的功能我觉得太超模了怎么办？**
A：丰俭由人！本模组为几乎所有新增功能都配置了独立设置开关，觉得不合适直接关掉即可。

**Q10：寸心精魄的获取概率是多少？**
A：凝结寸心时概率约从5.4%递增至58%。如果到第7点寸心仍从未获得过精魄，会触发首次保底获得1枚。

**Q11：骨刃的淬炼和其他武器有什么不同？**
A：骨刃每次淬炼只增加6点位面伤害（其他武器10点），但耐久上限会提升（最高400），且满阶后0耐久不碎裂，进入"枯竭态"保留位面伤害和吸血。

**Q12：《腌笃鲜》玉兔会覆盖《神话书说》原玉兔吗？**
A：不会。玉兔仙子是独立制作的重置人物，与《神话书说》原玉兔同时存在，角色身份、出生物品、曲谱、药效和人物进度互不覆盖。

**Q13：为什么玉兔开局只有《田中乐》，也不能召唤月宫守卫？**
A：其余九曲需要向嫦娥展示指定物品请教。学会第一首非初始曲目后解锁1名守卫；掌握曲目总数达到9首后，守卫上限直接提升到3名。

**Q14：为什么我在黑风大王死亡后释放吞天，没有出现熊罴真身？**
A：顺序反了。必须先完成吞天施法，让活动场覆盖黑风大王的死亡位置，再在场内真正击杀他。死亡后才补放、蓄力尚未完成、或只让遁风后来飘入范围，都不能触发。

**Q15：熊罴真身挑战失败后为什么下次生命反而降低了？**
A：动态生命读取本轮原黑风大王的真实击败次数。计数在原黑风死亡时就已经增加，即使当次没有成功截住遁风也不会撤销；失败返风同样不会清零。你必须重新击杀一次原黑风才能再次触发，所以生命会按24000、21000、18000、15000、12000逐级降低。真正击杀熊罴真身后才会归零。

**Q16：辟火罩为什么没有把范围内所有火直接扑灭？**
A：辟火罩的定位是“束焰”而不是全屏灭火。它让领域内玩家免受火焰伤害，暂停受控燃烧物的烧毁进程并阻止火势外传；每处受束火焰仍会独立灼伤近处目标，并在罩心附近更为强烈。领域收起或目标离开后，原燃烧会从剩余时间继续。
